Office Add-in#

Introduction#

This section details how you can use the office add-in to communicate with your Pyplan application.

Installation#

To add the Pyplan add-in, open the list of add-ins and use the search box. Then select Pyplan for Microsoft 365 and confirm the installation.

Login#

To log in to a Pyplan instance, you must first register that instance. To do this, follow the steps below:

  • Click on the Sign in button

  • If this is the first time you log in, you must register the instance you wish to connect to. To do this click on the + button and then fill in the instance data and click on confirm.

Connecting using SSO

If you are going to use SSO (Single Sign On) login, in the url you must add /saml/[company code]. Example: https://dev.pyplan.com/saml/pyplan

  • Once you register the connection, it will be available for future logins. Then click on the connection you wish to use and log in using your credentials.

Open an application#

  • After logging in, you will be able to open any of the applications you have available, either from your private space or from the public space or from a work team to which you belong.

Node Links#

Node Links are links between Microsoft Excel spreadsheets or ranges and Pyplan nodes. These links allow you to send and receive information to and from your Pyplan application.

Types of Node Links#

There are two types of Node Links:

  • data output: allow to obtain data from Pyplan

  • data input: allow to send data to Pyplan

To create a Node Link for data output perform the following steps:

  • Click on the + button.

  • Select the destination of the data. It can be in the selected cell or in a new worksheet. In case you select a new sheet, a sheet with the Pyplan node ID will be created.

  • Select the Pyplan node from which the data will be obtained. All nodes are data output nodes, except form type nodes.

To create a data entry Node Link perform the same steps as above, but select a form type node. This will allow you to modify the cell values and then send the data to Pyplan. To send the data to Pyplan once the table cells have been modified you must click on the send data button.

Sending data

When sending data to Pyplan, the Form node will be updated and if there are nodes in the Excel workbook that depend on the updated node, they will be updated with the new result. You can also make a pivot table based on the data output table. Updating the Node link will also update the pivot tables.
