Applications#

Application features#

A Pyplan application is a particular development to solve a specific problem, which uses the functionalities provided by the platform.

Pyplan has been used to develop tools such as: Demand Planning, Sales & Operations Planning, Financial Planning, Pricing, Budgeting Planning and Forecasting among many other applications.

An application in Pyplan can be composed of several elements, among them:

  • Reading external data sources.

  • Data input forms.

  • Calculation and processing modules.

  • Visualization interfaces.

These elements are stored in a workspace.

All users with application creation permissions have access to at least two workspaces:

  • A public space called “Public “ where applications accessible to all users are shared.

  • A private space called “My Workspace “, where each user can create or modify applications in an environment only accessible to him.

In addition, users have access to workspaces of teams to which they belong: “Teams “. These teams are the way to share information with restricted access to certain users.

Teams#

Work teams are created by the system administrator and access permissions are granted by “Department “ to which a user belongs. The same user can be part of several “Departments “
