Pyplan User Guide#

This guide is the main repository for the documentation on how Pyplan works. Its purpose is to explain what Pyplan is used for and how the platform works.

It is not the purpose of this guide to instruct on programming in Python or on the multiple Python libraries used by Pyplan, such as Pandas, Xarray, Numpy, Plotly, among others. Links to the specific documentation for these libraries are included in this guide to facilitate the user’s access to the relevant content for understanding some functionality of Pyplan.

Note

To understand how Pyplan works, it is necessary to understand what it is and what this platform is used for.

What is Pyplan?#

Pyplan is a software platform for creating and sharing planning and data analysis applications.

What is a planning and data analysis application?#

A planning and/or data analysis application is a software development designed to provide quantitative support for a decision-making process. An application of this type is basically constituted by a data injection process from multiple sources, a calculation process according to different business rules, and a visualization and analysis process of results.

What is the difference between a planning application and a data analysis application?#

A planning application is a calculation process that allows for projecting results according to specifically defined business rules. In a data analysis application, on the other hand, the business rules are unknown. These are inferred from the analysis of historical data using statistical or artificial intelligence processes.

How do I access Pyplan?#

Pyplan is a platform that is installed on a server that is accessed with the corresponding user credentials and password. You can read here <https://pyplan.com> about Pyplan licensing alternatives.
